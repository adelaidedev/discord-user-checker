How do I find a Discord User ID?

To find a Discord User ID, follow these steps:

  1. Open Discord and go to Settings.
  2. Navigate to Advanced and toggle on Developer Mode.
  3. Exit settings, go to any server, and type a message in a channel.
  4. Right-click on your username or any user's name in the chat, and select Copy ID.

You now have the Discord User ID copied to your clipboard!
